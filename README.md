# Eati Website

Landing site for [Eati](https://eatiapp.com) – AI Calorie Tracker. Built with Next.js (App Router).

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- **app/** – Pages, layouts, API routes (blog, tools, foods, contact)
- **components/** – React components (calculators, sections, Navbar, Footer)
- **content/blog/** – Blog articles (JSON)
- **lib/** – Utilities (blog, foods, analytics)
- **public/** – Static assets and images

## Deploy

The project can be deployed on [Vercel](https://vercel.com) or any Node.js host. Set environment variables (e.g. `OPENAI_API_KEY`, `NEXT_PUBLIC_SITE_URL`) in your deployment environment.
